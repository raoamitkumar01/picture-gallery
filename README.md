# 🖼️ Picture Gallery Website

This is a simple Picture Gallery Website made using HTML and CSS.

## 📌 About Project

This project shows multiple images in a gallery layout.

Images are arranged using grid / flexbox and styled using CSS.

This project is created for learning web design.


git hub - https://raoamitkumar01.github.io/picture-gallery/

## 🛠️ Technologies Used

- HTML
- CSS
- VS Code
- GitHub

## 📂 Files in Project

- index.html → main gallery page
- style.css → design file
- images → image folder

## ⚙️ Features

- More than 10 images
- Grid layout
- Hover effect
- Simple design
- Beginner friendly code
